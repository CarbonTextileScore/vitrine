Architecture de la solution

L'API

Le backend est construit en utilisant le framework Spring en Java pour créer une API REST.
Il utilise également Spring Boot pour faciliter la configuration et le déploiement, et Spring Security pour gérer l’authentification et l’autorisation.
De plus, il utilise JWT (JSON Web Tokens) pour la livraison et la vérification des tokens, dans le but de renforcer la sécurité des communications entre le client et le serveur.
